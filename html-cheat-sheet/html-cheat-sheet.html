HTML Anatomy
Component Description Example
HTML element A unit of content in an HTML document formed by HTML tags. <p>Hello World</p>
HTML tag the element name, surrounded by an opening (<) and closing (>) angle bracket. <p> ,</p>
Opening tag the first HTML tag used to start an HTML element. The tag type is surrounded by opening and closing angle brackets. <p>
Content The information (text or other elements) contained between the opening and closing tags Hello World
Closing tag the second HTML tag used to end an HTML element. Closing tags have a forward slash (/) inside of them, directly after the left angle bracket. </p>
self-closing tag may include or omit the final slash -

HTML Basic Tags
Tag Name Description
<!-- comment --> Comments.
<!DOCTYPE html> should always be the first line of code in your HTML files. This lets the browser know what version of HTML to expect.
<html> contain all HTML code.
<head> info about the web page, like the title.
<title> title of web page, appears in a browser's tab.
<a> Anchor tags, linking to other pages or content on the same page.
<body> Any visible content should be placed within.
<h1> ~ <h6> Headings and sub-headings, are used to provide titles for sections of content.
<p>,<span>,div specify text or blocks.
<em>,strong emphasize text.
<br> line break.
<ol> Ordered List, numbered.
<ul> Unordered List, bulleted.
<li> list item.
<img> self-closing tag. linking to an existing source.
required attribute src, the image source or location.
alt is description of the image.
<video> video source src.
the width and height attributes are used to set the size of the video displayed in the browser.
The text between the opening and closing video tags will only be displayed if the browser is unable to load the video.

HTML Table Tags
Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attribute
Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
Semantic HTML
Tag Description
<header> usually for either navigational links or introductory content.
<nav> define a block of navigation links such as menus and tables of contents.
<main> encapsulate the dominant content within a webpage.
<footer> the content at the bottom of the subject information.
<section> elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> holds content that makes sense on its own.
<aside> mark additional information.
figure encapsulate media such as an image, illustration, diagram, code snippet, etc
figcaption describes the media in the figure.
audio encapsulate a source element to link.
controls: the audio controls.
autoplay: literally.
embed self-closing tag.
can embed any media content incluing videos, audio files, and gifs
